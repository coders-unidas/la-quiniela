Introduction

The aim of this project is to develop a machine learning model simulating a Quiniela game using historical data from LaLiga. The model can give you the predictions of the results of the matches for a given match day in a certain division an season. For every match in that given day, the output can be: 1 if the home team wins, 2 if the away team wins or x if there is a tie.
